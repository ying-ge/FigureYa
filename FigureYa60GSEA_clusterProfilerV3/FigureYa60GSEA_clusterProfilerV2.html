JavaGSEADIYFigureYa13GSEA_Java

clusterProfiler is good at enrichment analysis, you can use GO, KEGG, Diseaes, Reactome, DAVID, MSigDB, and even custom annotation libraries to do enrichment analysis. enrichplots rich drawing functions cover almost all the ways of presenting enrichment analysis results, http://bioconductor.org/packages/devel/bioc/vignettes/enrichplot/inst/doc/enrichplot.html.

Suitable for: those who have done enrichment analysis with clusterProfiler and want to DIY the result plot by themselves.

If you want to use Java version of GSEA to do enrichment analysis and DIY result plot by yourself, please use FigureYa13GSEA_Java.
